# 제어문

## if문

**if statement**

```jsx
if ( 조건부분 ) {
동작부분
}
else
```

## else if문

조건이 2가지 이상일 때 `else if` 문 사용

## switch문

```jsx
switch ( 비교할_값 ) {
	case 조건값_1:
		동작부분;
		break; // switch문을 빠져나옴
	case 조건값_2:
		동작부분;
		break;
	default: // 비교할 값이 모든 조건값과도 일치하지 않을 경우. 생략가능
		동작부분;
}
```

조건과 일치하는 경우를 찾은 다음 **break문을 만나기 전까지** 그 아래 모든 동작을 실행

### switch문 vs. if문

- **범위**를 만족하는 조건식을 만들 때 ⇒ **if문**이 효과적
- **특정값**을 만족하는 조건식을 만들 때 ⇒ **switch문**이 효과적

**switch문을 if문으로 바꿀 때**

🚨 **주의** : switch문은 값들을 비교할 때 자료형을 엄격하게 구분 (암시적 형 변환을 허용하지 않음) 

if문으로 대체할 때, 조건식에서 반드시 **등호 3개를 사용**해 일치 비교를 해야 함

## for 반복문

**for statement**

```jsx
for (초기화부분; 조건부분; 추가동작부분){
	동작부분;
}
```

- **초기화 부분**: for반복문이 시작할 때, 가장 먼저 딱 한 번만 실행되는 부분
    
    주로 반복 횟수를 저장할 변수를 생성하는데 활용
    
- **조건 부분**: true면 동작부분 실행
- **추가동작 부분**: 추가적으로 동작할 부분

### for 반복문 Tip

1. **추가 동작 부분**을 꼭 채울 필요는 없다

```jsx
for (let i=1; i<=10) {
	console.log(`${i} 코드잇 최고!`);
	i++;
}
// 추가 동작 부분을 비워두고 동작 부분에 i를 1씩 증가시키는 부분이 들어감
// but, 동작부분에서는 반복하고자 하는 내용만 작성하는 것이 좋다
```

1. 초기화 부분에서 생성한 변수는 **for문의 로컬 변수**다
    
    → for 반복문 종료 후 for 반복문 밖에서 변수 사용하려고 하면 오류 발생!
    

1. **초기화 부분**도 반드시 채울 필요는 없다
    
    for 반복문의 로컬 변수를 꼭 활용할 필요 없이, 반복문 밖에 있는 글로벌 변수를 활용해도 아무런 문제 x
    
    ```jsx
    let i = 1;
    for(**;** i<=10; i++){
    	console.log(`${i} 코드잇 최고!`);
    }
    ```
    
    단, for 문의 **소괄호 안쪽 가장 첫번째 세미콜론은 생략할 수 없다.**
    
    ⇒ for문의 **소괄호 안쪽은 반드시 세미콜론 2개가 필요!**
    

### for 반복문 실습

주어진 높이( height )에 맞게 *로 삼각형을 그리는 함수 printTriangle()을 완성

```jsx
function printTriangle(height){
	let msg='';
	for(let i=1; i<=height; i++){
		msg = msg + '*';
		console.log(msg);
	}
}

printTriangle(1);
printTriangle(3);
printTriangle(5);
```

## while 반복문

```jsx
while(조건부분){
	동작부분
}
```

- 초기화 부분이 없음 → 반복문 밖에서 글로벌 변수를 미리 만들어 반복에 필요한 횟수를 카운트
- 조건 부분은 for문과 동일
- 추가동작부분이 없음 → 동작부분에서 반복 횟수를 증가시키는 부분을 작성

글로벌 변수를 조건 비교하여 사용하고 반복문 내부에서도 다루면서 **반복문이 종료된 다음에도 변수를 사용**해야 될 때 ⇒ **for문보다 while문을 쓰는 것이 좋음!**

## break와 continue

### break

반복문의 조건 부분과 상관없이 반복이 실행되는 도중 빠져나갈 수 있음

반복문 전체를 빠져나옴

### continue

```jsx
//1부터 10까지 숫자 중에서 홀수만 출력
//for문
for (let i = 1; i <= 10; i++) {
	if (i % 2 === 0) {
		continue; // 밑의 console.log(i)실행하지 않고 추가동작부분으로 넘어감
	}
	console.log(i);
}

//while문
let i = 1;
 
while(i <= 10) {
	if (i % 2 === 0) {
		i++;
		continue;
	}
	console.log(i);
	i++;
}

```

**동작부분을 한 번 건너뜀**

continue를 만나면 **그 아래 코드들은 실행되지 않고 바로 다음 단계로 넘어감**

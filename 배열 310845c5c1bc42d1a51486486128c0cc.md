# 배열

## 배열

**Array**

여러 값들을 **쉼표**로 구분하여 순서대로 **대괄호** 안에 나열

- **요소(element)**: 배열 안의 값
- **인덱스(index)**: 각 요소의 순서
    - 객체에서의 프로퍼티 네임의 역할
    - 0부터 시작
- **인덱싱(indexing)**: 인덱스를 통해 요소에 접근
    - `배열이름[index]`

## 배열 다루기

배열의 자료형은 **객체**이다. → 여러 메소드 사용가능

### length

배열 안에 들어있는 요소가 총 **몇개**인지 표시

`배열이름.length`

`배열이름['length']`

배열의 **마지막 요소**에 접근

`배열이름[배열이름.length - 1]`

### 요소 추가&수정

**할당연산자** 사용

- **존재하지 않는** 요소에 접근해서 새로운 값 할당 → 배열에 요소 **추가**
- **이미 존재하는** 요소에 접근해서 값을 할당 → 요소 **수정**

📌 **인덱스의 순서를 뛰어넘어서** 요소를 추가 ⇒ **undfined** 값을 가진 요소가 자동 생성

### 요소 삭제

**`delete` 연산자**

값 자체는 지워지지만 요소가 empty로 남아있음 length도 유지

완벽한 삭제는 아니다

## 배열 메소드

### **`splice` 메소드**

**요소를 완벽하게 삭제**하기 위해 사용

삭제하고 싶은 인덱스를 파라미터로 전달

`arr.splice(삭제할index)` **시작한 인덱스 이후 모든 요소 삭제**

`arr.splice(삭제할index, 삭제할개수)` 삭제를 시작한 인덱스로부터 **삭제할 개수만큼 삭제**

`arr.splice(삭제할index, 삭제할개수, 추가할요소, 추가할요소, …);`

- **삭제할개수를 0으로** 설정 → 아무것도 삭제하지 않고 배열안에 어느곳이든 **요소 추가** 가능
- 삭제할 개수만큼 요소를 추가 → **요소 수정**

🚨 **주의**: **반복문을 통해 요소 삭제시 index가 밀림**

```jsx
// numbers 배열의 요소들 중 홀수를 모두 삭제해 주세요.

//방법1
for (let i = 0; i < numbers.length; **i++**) {
  if (numbers[i] % 2 !== 0) {
    numbers.splice(i, 1);
    **i--; //요소가 삭제될 때 i를 미리 한 번 감소**
  }
}

//방법2
for(let i = 0; i < numbers.length; ){
  if ( numbers[i] % 2 !== 0)
    numbers.splice(i,1);
  else
    i++; **// 삭제되지 않을 때만 i를 증가**
}
```

### **배열의 양 끝 요소를 다루는 메소드**

- **배열의 첫 요소를 삭제**

`arr.shift()`

- **배열의 마지막 요소를 삭제**

`arr.pop()`

- **배열의 첫 요소로 값 추가**

`arr.unshift(value)`

- **배열의 마지막 요소로 값 추가**

`arr.push(value)`

### 다양한 메소드

- **배열에서 특정 값 찾기:** `array` 배열에 `item`이 포함되어 있는지 확인
    
    `array.indexOf(item)` : **앞에서부터** 탐색
    
    - 만약 포함되어 있다면, **`item`이 시작되는 인덱스**가 리턴됩니다.
    - 포함되어 있지 않다면, **`-1`**이 리턴됩니다.
    - 여러 번 포함되어 있으면, 처음 발견된 인덱스가 리턴됩니다.

`arr.lastIndexOf(item)`: **뒤에서부터** 탐색

```jsx
let brands = ['Google', 'Kakao', 'Naver', 'Kakao'];

console.log(brands.indexOf('Kakao')); // 1 출력
console.log(brands.indexOf('Daum')); // -1 출력 
console.log(brands.lastIndexOf('Kakao')); // 3 출력
console.log(brands.lastIndexOf('Daum')); //-1 출력
```

- **특정 값이 배열 안에 있는지 여부만 확인**
    
    `array.includes(item)`
    
    `array`배열에 `item`이 있을 경우 `true`를, 없을 경우 `false`를 리턴
    
    ```jsx
    let brands = ['Google', 'Kakao', 'Naver', 'Kakao'];
    
    console.log(brands.includes('Kakao')); // true 출력
    console.log(brands.includes('Daum')); // false 출력
    ```
    
- **배열 뒤집기**
    
    `array.reverse()`
    
    ```jsx
    let brands = ['Google', 'Kakao', 'Naver', 'Kakao'];
    console.log(brands); // (4) ["Google", "Kakao", "Naver", "Kakao"] 출력
    
    brands.reverse();
    console.log(brands); //(4) ["Kakao", "Naver", "Kakao", "Google"] 출력
    
    ```
    

## for…of 반복문

```jsx
**for** (변수 **of** 배열) {
동작;
}
//변수 부분에 **배열의 요소**가 할당
```

- **객체**
    
    **for…in** 반복문 
    
    변수에 프로퍼티 네임(key)가 할당됨
    
- **배열**
    
    **for…of** 반복문
    
    변수에 배열의 요소가 할당됨
